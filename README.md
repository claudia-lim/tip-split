<h1>Tip Splitter App</h1>

An app that allows you to enter a bill amount, tip percentage and number of people which then calculates the total tip and amount per person.

<h2>Built with:</h2>
<ul>
  <li>Javascript</li>
  <li>React</li>
  <li>CSS Grid and Flexbox</li>
</ul>
